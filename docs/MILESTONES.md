# Milestones

## M1: 打通最小链路（1~2 天）
- 扩展读取当前歌曲与播放进度
- 本地服务接收事件并打印
- 悬浮窗能显示静态文本

## M2: 同步歌词（2~4 天）
- 接入至少 1 个歌词源
- 实现时间轴高亮
- 切歌/暂停/拖动可正确同步

## M3: 稳定化（3~5 天）
- 多 provider + 缓存 + 降级
- 指标采样与日志
- 错误恢复与自动重连

## M4: 低占用优化（2~3 天）
- 合并事件、降低刷新率
- 清理多余对象与缓存策略
- 压测并调优内存峰值

## M5: 体验增强（可选）
- 快捷键显示/隐藏
- 多主题与字体配置
- 双语歌词/翻译模式
